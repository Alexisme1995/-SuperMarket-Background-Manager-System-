# SupermarketManagerSystem
一个精简的超市后台管理系统，麻雀虽小，五脏俱全


前端使用纯HTML代码开发，无框架


后端使用框架：Struts2+Spring+Hibernate




扩展插件：DWR支持，JSTL，日历插件，时间插件，sweetAlert插件，弹窗插件



风格:简约



包含功能：



用户权限设置：分为经理管理员(0)和普通用户组(1),(3)代表只能对自己的数据进行操作



登录
